position: sticky;

親要素の範囲内でスクロールに応じて位置が変わる。ついてきてあるべきところに落ち着くイメージ。

スクロール

min(変動値, 固定値);　max(変動値, 固定値);　clamp(最小値、推奨値、最大値);

min(変動値, 固定値); 最大が1000px

max(変動値, 固定値); 最小が1000px

clamp(最小値、推奨値、最大値);

calc(計算式)

          
        //親要素が375pxで、子要素が、300pxの時、widthには80%と設定したい。
        .sample{
          width: calc(300 / 375 * 100%);
        }
        //親要素が375pxで、子要素が、300pxの時、widthには80%と設定したい。最大値は、500pxとしたい。
        .sample{
          width: min(calc(300 / 375 * 100%), 500px);
        }
        //親要素が375pxで、子要素が、300pxの時、widthには80%と設定し、
        　最大値は、500px、最小値は300pxとしたい。
        .sample{
          width: clamp(300px, calc(300 / 375 * 100%), 500px);
        }

@media (width >= 〇〇px)

書き方がわかりやすくなった。
@media (width >= 375px <= 1024px)
375px以上1024px以下の時に適用される。以下と同じ。
@media (min-width: 375px) and (max-width: 1024px)

transform

プロパティ 説明
translate 要素を水平方向や垂直方向に移動する。
rotate 要素の回転を指定する。
scale 要素をx軸、y軸方向に拡大・縮小する。
animation 要素にアニメーションをつけることができる。
@keyframes アニメーションの開始から終了するまでの間で、どのような変化をさせるかを指定する。
